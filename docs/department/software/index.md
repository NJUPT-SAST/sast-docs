## 🤔 关于我们

软件研发部是校科协的核心部门之一，我们致力于为对软件开发💻、算法设计🧮、安全🛡️等领域感兴趣的同学提供一个学习、交流和实践的平台。部门下设 **C++ 组**、**C#组**、**Python 组**、**Web 研发组**、**运维组**、**算法组**、**Graphics 组**、**游戏开发与设计组**和**安全组**，涵盖了软件相关的各个方向。

即使你是技术小白也完全不用担心！软件研发部会举办**方草地公开课**，帮助你入门编程，各组也会开展授课，循序渐进地提升你的编程能力。如果你有任何问题，也可以到**大学生活动中心 101** 来和学长进行交流，一起解决问题。

加入软件研发部，不需要你是编程大神，只需要你怀揣着对技术的热情🔥和学习的渴望🌟！加入我们，来认识更多技术爱好者，学习前沿技术，并不断钻研，深入，精进！

**软件研发部欢迎任何对技术有着充分热情的同学加入！🎉**

## ⛄ 建议与帮助

下面是一些软件开发方向的学习建议。

如果未来想继续走计算机相关方向（考研除外），下面是一些忠告：

1. **学英语，用英语**，尤其是计算机相关的表达。不要滥用全文翻译，这对学英语没有帮助。可以使用划词翻译。推荐使用浏览器插件：**沉浸式翻译**，总的来说，推荐大家努力去尝试适应全英文上网，可以先从只使用英文关键词来搜索开始。

2. **不要使用百度**，可以使用 Google、Duckduckgo 或者 Bing（国际版）。

3. **尽可能减少使用 CSDN 等野生的中文互联网资料**，上面的许多内容过时、片面甚至错误。如果觉得 CSDN 好，可能只是见识太少了而且不会英语，没有见过更优质的内容。（参考实验报告可以用）

4. **不提供恰当的资助就不要随便向陌生人问低级问题**，所有人都很忙而且没有回答的义务。可以先去问搜索引擎，如果得不到解决，请携带上下文（参考 [提问的智慧](https://github.com/ryanhanwu/How-To-Ask-Questions-The-Smart-Way)）和**截图**（不是拍屏）询求帮助。

5. **使用 AI 大模型时，注意信息甄别**，不建议无脑询问可以直接从一般搜索引擎上得到准确答案的问题。（如：「南京邮电大学的邮政编码是多少」）
