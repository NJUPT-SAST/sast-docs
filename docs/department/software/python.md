## Python 是什么？

~~Python 是一款诞生于荷兰国家数学与计算机科学研究中心的 Guido van Rossum 之手的编程语言，最初作为BASIC语言的简单替代品的替代品。Python在各个编程语言中比较适合新手学习，且随着版本的不断更新和语言新功能的添加,逐渐可被用于独立的、大型项目的开发。~~

Python 是由荷兰程序员 **Guido van Rossum** 发明的一门古老而优雅的通用编程语言。程序运行在一个被称作「**Python 解释器**」的神秘空间里，在这里，被智慧之光眷顾的人将被授予「**动态类型**」的力量，驾驭代码的灵活性。你将扮演一位名为「**开发者**」的智慧角色，在自由的编码旅程中邂逅性格各异、能力独特的解释器，和它们一起消灭错误，优化性能——同时，逐步揭开「**GIL（全局解释器锁）**」的真相。

> 💡 **PS**: GIL 在 python 最新最热的 3.14 发布中已集成了关闭方式

## Python 能做什么？

Python 放弃了部分性能转而追求**易学易用**和**强可拓展性**，大部分的功能都有对应的第三方库和文档，使得 Python 在计算机领域得到了广泛的应用，如作为胶水语言连接其他语言或者快速实现demo等。

在 SAST，你可以探索目前 Python 最热门的方向，包括：

- 🤖 机器学习
- 🕷️ 爬虫

想了解更多和 Python 组方向相关的详细学习内容，请继续阅读下文。

### 在本组学习 Python 开发的建议

- **行业现状**：Python 开发在某些领域（如 Web 开发、数据分析、机器学习）非常热门，但在其他领域（如高性能计算、嵌入式开发）可能不是首选。就业面相对较广，但竞争也较为激烈。

- ⚠️ **不要用应试思维学习 Python 开发**，刷题对提升开发能力帮助有限。最有用的学习方式是：以实际项目需求为原动力，在编写代码的实践中学习，通过查找资料、调整方案、解决问题的流程来提升开发水平，积累项目经验。

- 📚 课堂内容严重脱节于实际生产，如果想从事 Python 开发，请务必拥有**自行查找资料并自学的能力**。网络上 Python 方向的免费学习资料非常多，学会查找、筛选、学习、应用。

- 💼 计算机大类专业（计算机、通信、自动化、物联网）并不会限制你的就业方向，以自己的兴趣为导向，才能让自己有动力学习。

- 🎯 我们的授课内容并不是你要学习的全部知识，更多地是**提供学习指引**，告诉你往什么方向学习。
