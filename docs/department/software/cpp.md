## 3WH Questions

### C++ 是什么？

C++ 是由贝尔实验室自主研发的一门古老的通用编程语言。程序运行在在一个被称作「C++ 抽象机」的机器上，在这里，被神选中的人将被授予「`auto`」，导引模板之力。你将扮演一位名为「程序员」的神秘角色，在自由的编码过程中邂逅性格各异、能力独特的编译器，和他们一起消灭 bug，找回泄露的内存——同时，逐步发掘「UB」的真相。

### 学习 C++ 能做什么？

C++ 语言的地位有些特别，它在计算机的不同领域均有所涉及。而在 SAST，你可以在不同部门学习到不同的方向：

- **C++组**：探索语言本身，包括但不限于语法，框架、库；学习 PC 客户端（Qt 等）和后端（网络编程）开发
- **游戏组**：游戏开发（UE 方向）
- **Graphics 组**：图形开发
- **算法组**：算法方向
- **电子部**：嵌入式、计算机视觉
